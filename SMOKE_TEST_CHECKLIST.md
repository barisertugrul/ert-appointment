# ERT Smoke Test Checklist

Bu dosya, admin menüsündeki **ERT Smoke Test Checklist** bağlantısı için uyumluluk amacıyla eklenmiştir.

Güncel checklist dokümanı:
- `docs/pro-whatsapp-smoke-checklist.md`

Kısa kontrol adımları:
1. Frontend formunda randevu oluşturma akışını test et.
2. Yönetim panelinde randevu onay/iptal akışını test et.
3. Bildirim tetiklerini (e-posta/SMS/WhatsApp) doğrula.
4. Dashboard listeleri ve sıralamaların beklenen şekilde çalıştığını doğrula.
5. Pro özelliklerin lisans durumuna göre doğru göründüğünü kontrol et.
